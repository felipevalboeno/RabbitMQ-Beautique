# RabbitMQ + Spring: Beauty Salon Project

In this project, I worked with Microservices and RabbitMQ to simulate a beauty salon management. All the development was doinh use Java SpringBoot Framework. 

Folders:

 - beautique: The source code project
 - db_beautique: data persistence
 - infrastructure: docker-compose and schema SQL.

## Technologies:
   - Java 17
   - Maven
   - PostgreSql
   - Flyway: To create some migrations and to change the not null configutations of some fields in database.
